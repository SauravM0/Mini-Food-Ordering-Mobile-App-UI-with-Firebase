# Theme System

This document explains the theme system used in the Mini Food Ordering App.

## Color Palette

The app uses a warm, food-friendly color palette:

- **Primary**: `#FF6B35` (Warm orange) - Used for primary buttons and highlights
- **Secondary**: `#C1440E` (Rich red) - Used for secondary buttons and badges
- **Background**: `#FFF9F5` (Light cream) - Main background color
- **Surface**: `#FFFFFF` (White) - Cards and container backgrounds
- **Text**: `#331E11` (Dark brown) - Main text color
- **Muted Text**: `#8B7365` (Medium brown) - Secondary text
- **Border**: `#E5DED8` (Light grayish brown) - Dividers and borders
- **Success**: `#4CAF50` (Green) - Success messages
- **Danger**: `#F44336` (Red) - Error messages and destructive actions

## Typography

The typography system defines four text roles:

- **Heading**: 24px, bold - Screen titles
- **Subheading**: 18px, semi-bold - Section headers
- **Body**: 14px, regular - General text content
- **Small**: 12px, regular - Labels and secondary information

## Spacing

The spacing system uses consistent tokens:

- **xs**: 4px - Small elements and text lines
- **sm**: 8px - Related components
- **md**: 16px - Standard spacing
- **lg**: 24px - Section separation
- **xl**: 32px - Screen edges and major spacing

## Border Radius

- **Standard**: 12px - Cards and primary buttons
- **Light**: 6px - Small badges and inputs

## Shadows

- **Card**: Subtle shadow for depth and separation
